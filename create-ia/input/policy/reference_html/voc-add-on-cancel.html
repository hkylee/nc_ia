VOC analysis readout

부가서비스 해지 VoC 분석

부가서비스 해지 VoC의 핵심은 단일 기능 부족보다 고객이 처리 조건, 영향, 결과를 스스로 확인하지 못해 상담으로 전환되는 데 있다.
가장 큰 병목은 해지 후 요금 재청구 혼란이며, 통합채널은 이를 탐색, 판단, 실행, 확인, 복구가 연결된 업무 경험으로 재설계해야 한다

1. 핵심 신호

부가서비스 해지 VoC는 고객이 업무를 시작하고, 조건을 이해하고, 결과를 확인하고, 실패를 복구하는 전체 흐름을 정책 기준으로 재설계해야 함을 보여준다.

Core signals
01

해지 후 요금 재청구 혼란

" 해지했는데 왜 또 나오냐 " → 처리 결과에 대한 즉각적 확인 욕구 + 불신 해소 욕구

02

오가입 즉시 해지 경로 부재

" 실수로 가입됐는데 요금이 나갈까봐 불안 " → 즉각적 통제감 회복 욕구

03

부가서비스 현황 확인 및 해지 경로 불명

" 내 폰에 어떤 서비스가 켜져 있는지 한눈에 보고 직접 관리하고 싶다 " → 현황 파악 및 자율 관리 욕구

04

공휴일 · 야간 해지 처리 불가

" 지금 당장 해지하고 싶은데 공휴일이라 안 된다는 답답함 " → 24 시간 셀프 처리 채널 욕구

05

모르는 사이 과금 및 반복 재가입

" 모르는 사이 가입되는 것을 막고 싶다 " + " 이미 빠져나간 요금을 돌려받고 싶다 " → 신규 가입 즉시 알림 및 차단 설정, 환불 신청 기능 욕구

3. Hidden Needs

표면 요청 뒤에는 고객이 직접 판단하고 처리 결과를 신뢰하고 싶은 요구가 숨어 있다.

Hidden needs
안심

즉각적 통제감 회복 욕구 (1 위): " 실수로 가입됐는데 요금이 나갈까봐 불안

즉시 취소하고 싶다 " → 가입 완료 문자 원터치 해지 딥링크 + 앱 즉시 취소 버튼 대응 필요

처리가 제대로 됐는지 확인하고 싶다 " → 해지 완료 화면 잔여 청구 안내 + 해지 이력 실시간 조회 대응 필요

즉시

현황 파악 및 자율 관리 욕구 (3 위): " 내 폰에 어떤 서비스가 켜져 있는지 한눈에 보고 직접 관리하고 싶다 "

부가서비스 현황 대시보드 + 일괄 해지 기능 대응 필요

연계

신규 가입 즉시 알림 + 차단 설정 + 환불 신청 기능 대응 필요

아래 계기는 고객이 셀프 채널을 이탈해 상담으로 전환하는 실제 출발점이다.

  5. 05앱에서 부가서비스 해지 방법 몰라 전화 — 전 기간 지속

  18. 18유심 교체 · 기기변경 후 유심 보호 서비스로 인식 불가 — 전 기간 지속

5. Quick Win

즉시 개선 가능한 항목은 진입점 노출, 완료 안내, 상태 조회, 권한 처리, 실패 복구를 중심으로 정리된다.

6. 요구사항·정책서 전환 질문

VoC 분석 결과는 부가서비스 해지 정책서에서 아래 질문에 답할 수 있는 기능과 정책 항목으로 전환되어야 한다.

Policy conversion
사전 고지

무엇을 처리 전에 알려야 하는가?

요금, 혜택, 적용 시점, 제한 조건, 후속 영향을 업무 실행 전에 이해 가능한 문장과 비교 정보로 제공해야 한다

상태·오류

실패와 오류를 어떻게 복구하는가?

접수, 검증, 반영, 실패, 취소 상태를 고객에게 보여주고 재시도, 대체 경로, 상담 전환 기준을 연결해야 한다

진입점

고객은 어디에서 업무를 시작할 수 있는가?

통합채널의 홈, 마이, 요금, 상품 영역에서 같은 업무 진입점을 제공하고, 현재 상태에서 가능한 다음 행동을 명확히 보여줘야 한다

처리 가능 범위

언제 셀프 처리할 수 있고 언제 제한되는가?

평일, 야간, 주말, 공휴일, 청구 마감, 개통·해지 시점별 처리 가능 범위와 대체 행동을 정의해야 한다

확인

처리 결과를 어떻게 확인시키는가?

처리 결과, 이력, 알림, 예상 청구 또는 사용 가능 상태를 고객이 직접 확인할 수 있어야 한다

연계

어떤 업무를 한 번에 묶어야 하는가?

상담, 앱, BSS, 외부 기관에서 끊어진 처리를 고객 과업 기준의 단일 흐름으로 재구성해야 한다

상태

처리 상태를 어디까지 보여줘야 하는가?

부가서비스 해지 업무의 접수, 진행, 완료, 실패, 취소 상태를 고객이 직접 확인하도록 정의해야 한다
