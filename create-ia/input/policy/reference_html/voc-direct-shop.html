1. 핵심 신호

다이렉트 샵 / 티월드 다이렉트 VoC는 고객이 업무를 시작하고, 조건을 이해하고, 결과를 확인하고, 실패를 복구하는 전체 흐름을 정책 기준으로 재설계해야 함을 보여준다.

Core signals
01

다이렉트 요금제 가입 조건 불명 및 변경 불가 안내 부재

"다이렉트 요금제가 더 저렴한데 왜 나는 못 쓰나 " → 정보 비대칭으로 인한 박탈감 + 비용 최적화 욕구

02

앱 / 웹 내 메뉴 탐색 실패 및 셀프 처리 경로 부재

" 직접 처리하고 싶은데 방법을 모르겠다 " → 셀프 처리 욕구와 경로 탐색 실패로 인한 답답함 해소 욕구

03

시스템 오류 및 인증 실패

" 오류를 빠르게 해결하고 싶다 " → 서비스 중단에 대한 불안감 해소와 셀프 처리 욕구

04

요금 · 결제 내역 및 서비스 구조 이해 불가

" 내가 내는 요금이 무엇인지 명확히 알고 싶다 " + " 더 저렴한 요금제를 찾고 싶다 " → 정보 이해 욕구와 절약 욕구

05

주말 · 야간 · 공휴일 셀프 처리 불가

" 지금 당장 처리하고 싶은데 주말 · 야간이라 막힌다 " → 24 시간 셀프 처리에 대한 즉시성 욕구
