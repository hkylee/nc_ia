# T멤버십 공개웹 보조지식: 등급·포인트·회원자격

작성일: 2026-05-03
자료 성격: 공개웹 보조지식
적용 주제: 멤버십 혜택, 멤버십 카드 관리, T 플러스 포인트, 미션 프로그램, 상품·서비스 혜택 이용/공유, 통합 쿠폰/이용권함, 회원 가입/탈퇴

## 1. 사용 원칙

이 문서는 T멤버십 공개웹에서 확인되는 등급, 포인트, 회원자격, T우주 LITE 전환 조건을 NC 정책서 작성에 참고하기 위한 보조자료다.
요구사항, 첨부 참고자료, 템플릿, 샘플 정책서와 상충하면 첨부 자료를 우선한다.
특정 연도 등급표와 이벤트 조건은 정책값 확정 근거가 아니라 판단축 후보로만 사용한다.

## 3. 정책서 작성 시 참고할 업무 관점

멤버십 등급은 통신요금 납부 금액, 가입 기간, 연체 이력, 요금제, 회선 정지 일수, 신규가입·명의변경 여부 같은 조건으로 산정된다.
멤버십 등급은 단순 표시값이 아니라 혜택 대상, 쿠폰 발급, 포인트 사용, VIP 전용 혜택, 등급 복원 기준에 영향을 준다.
T우주 LITE는 SK텔레콤 이동전화 고객이 아니어도 특정 조건에서 가입할 수 있는 별도 회원 유형으로 볼 수 있으며, T멤버십과 T우주 사이의 포인트, 이용이력, 탈퇴 유예, 전환 기준을 함께 고려해야 한다.

## 4. 후보 판단축

### 등급 산정

- 연간 납부 금액, 가입 기간, 연체 횟수, VIP 대상 요금제, 회선 정지 일수, 명의변경 여부를 정책 항목으로 분리한다.
- 등급 산정 기준일, 적용일, 복원일, 조정 횟수 제한을 별도 기준으로 둔다.
- 등급 산정에서 제외되는 금액 또는 서비스 항목이 있다면 별도 정책 항목으로 둔다.

### 포인트·쿠폰·혜택 사용

- 포인트 사용 가능 제휴처, 사용 단위, 적립/사용 구분, T우주 결제 사용 가능 여부를 분리한다.
- 쿠폰은 발급 조건, 사용 기간, 사용처, 양도 가능 여부, 재발급 가능 여부, 미사용 처리 기준을 둔다.
- T day, VIP PICK, PLUS 혜택은 기간, 대상, 수량, 등급, 중복 적용, 앱 사용 여부를 분리한다.

### 회원자격·전환

- T멤버십 회원, T우주 LITE 회원, 이동전화 해지 고객, 타사 이동전화 고객은 자격 기준과 이용 가능 서비스가 다르다.
- 탈퇴 유예, 포인트 소멸, 이용내역 보관, 재가입 또는 전환 시 이관 기준을 둔다.
- 선불폰, 택시폰, 타인명의 부당신청 등 가입 제한 조건은 정책 항목으로 관리한다.

## 5. Agent 사용 제한

연도별 등급표, 이벤트 혜택, 특정 제휴처 조건은 작성 시점에 변동될 수 있으므로 확정 정책값으로 쓰지 않는다.
정책서에는 등급 산정 구조, 혜택 사용 조건, 포인트/쿠폰 이력, 회원 전환 상태를 중심으로 반영한다.
