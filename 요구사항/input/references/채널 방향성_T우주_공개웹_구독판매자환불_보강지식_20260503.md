## 3. 정책서 작성 시 참고할 업무 관점

T우주는 구독형 상품을 고객에게 직접 판매하거나 판매자와 고객 간 거래를 중개할 수 있는 구독 플랫폼이다.
상품 유형은 쿠폰/할인형, 이용권형, 정기배송형처럼 사용 방식과 환불 기준이 다른 유형으로 나뉠 수 있다.
정책서에서는 회사, 판매자, 제휴사, 고객의 책임 경계를 명확히 하고, 상품 유형별로 가입, 결제, 사용, 변경, 해지, 환불, 고지 상태를 나누어야 한다.

구독 상품은 정기결제 주기, 무료체험, 혜택 소진, 쿠폰 발급, 이용권 등록, 배송 시작, 판매 종료, 채널 제한, 약관 변경 고지 같은 조건이 함께 작동한다.
판매자가 제공하는 상품은 상품 정보 변경, 판매 중단, 이용 불가, 제휴사 장애가 발생할 수 있으므로 고객 고지와 책임 주체, 대체 처리 기준이 필요하다.

## 4. 후보 판단축

### 가입·결제

- 구독 가입 가능 여부는 회원자격, 본인확인, 결제수단, 상품 판매 상태, 채널 제한, 제휴사 이용 가능 여부를 함께 본다.
- 정기결제는 결제일, 결제 실패, 재시도, 유예, 자동 해지, 혜택 중지 기준을 분리한다.
- T멤버십 포인트 또는 쿠폰을 결제에 사용하는 경우 사용 가능 상품, 사용 단위, 환불 시 복원 기준을 둔다.

### 사용·변경·해지

- 쿠폰/할인형, 이용권형, 정기배송형은 사용 시작 기준과 사용 완료 기준이 다르다.
- 고객 변경 요청은 결제 주기 전 변경, 다음 회차 반영, 즉시 반영, 변경 불가로 나눌 수 있다.
- 해지는 즉시 해지, 다음 결제일 해지, 혜택 유지 후 종료, 사용 이력에 따른 환불 제한으로 나누어 정의한다.

### 환불·책임 경계

- 환불은 상품 유형, 사용 여부, 혜택 소진 여부, 정기결제 시점, 판매자 정책, 법정 청약철회 기준에 따라 다르게 판정한다.
- 직접 판매 상품과 중개 판매 상품은 환불 책임과 고객 안내 주체가 다를 수 있다.
- 판매 종료 또는 상품 정보 변경 시 고객 고지, 대체 상품 안내, 기존 가입자 처리 기준을 둔다.

## 5. Agent 사용 제한

구독 상품별 실제 환불 금액, 판매자별 세부 약관, 프로모션 기간은 공개웹 자료만으로 확정하지 않는다.
정책서에는 구독 상품 유형, 결제·해지·환불 판단 기준, 책임 주체, 고객 고지, 이력 저장 기준을 중심으로 반영한다.
